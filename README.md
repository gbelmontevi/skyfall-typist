# Skyfall Typist

Skyfall Typist is a browser-based typing game where letters or short words fall from the top of the screen and must be typed before they hit the floor.

## Features

- Letter mode and word mode
- Progressive highlight feedback as you type a matching target
- Score, lives, clears, and level tracking
- Difficulty scaling over time
- Miss and clear animations
- Static-file friendly for GitHub Pages

## How to Play

1. Open `index.html` in a browser.
2. Click `Start Game`.
3. Type the falling letter or word before it reaches the floor.
4. If a target hits the floor, you lose a life.
5. Clear enough targets to increase the level and speed.

## Controls

- `A-Z`: Type letters
- `Backspace`: Remove the last typed character
- `Escape`: Clear the current typed input

## Project Files

- `index.html`: Main page structure
- `styles.css`: Game layout, visuals, and animations
- `script.js`: Game logic and keyboard handling

## Deploy To GitHub Pages

1. Push the repository to GitHub.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select your branch and `/ (root)`, then save.

GitHub will publish the game as a static site.
